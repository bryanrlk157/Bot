# SAFIRA PRO

## PASSO 1
Abra config.js e cole:

- TOKEN do bot
- ID do cargo
- ID do canal de auth
- ID do canal de logs

## PASSO 2
npm install

## PASSO 3
node index.js

## Railway
Basta subir os arquivos e rodar npm install automaticamente.

Sistema inclui:
- Botão fixo
- Banco JSON
- Logs
- Anti-bot (conta nova)
